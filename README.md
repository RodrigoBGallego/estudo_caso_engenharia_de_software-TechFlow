# Estudo de caso para a disciplina de Engenharia de Software

Software para controle de fluxo para a Falcão Logística

A empresa pediu um programa de controle de fluxo de trabalho e acompanhamento em tempo real. O software deve contar um quadro Kanban com acompanhamento em tempo real, alocação de colaboradores nas tarefas, edição de prioridade por tarefas e edição do escopo do projeto.
Além disso, a empresa pediu que cada perfil de colaborador tenha seus principais talentos, e que seja possível colocar as habilidades necessárias nas tarefas para que cada colaborador seja alocado nos projetos e tarefas tenham mais afinidade.
Esse software terá um programa de autenticação em 2 fatores, com o acesso do colaborador sendo liberado apenas para a empresa, e todo o software será feito em python, mas será acessado por aplicativo no celular e num site que a empresa deixará disponível apenas para os funcionários.
ATUALIZAÇÃO: Em reunião, foi decidido que o banco de dados seja em SQL. A estrutura do banco foi decidida em Colaboradores, Projetos, Tarefas. O banco foi definido como prioritário.
